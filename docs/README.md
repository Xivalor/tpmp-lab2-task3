# tpmp-lab2-task3

## Overview
Программа для работы с базой данных людей. Реализует ввод, сортировку и поиск записей по дате рождения.

Структура записи:
- Фамилия и инициалы
- Номер телефона
- Дата рождения (год, месяц, день)

## Author
Xivalor

## Usage
1. Сборка программы: `make`
2. Запуск: `./program`
3. Ввод данных: программа запросит 9 записей, затем покажет отсортированный список и спросит месяц для поиска
4. Очистка: `make clean`
